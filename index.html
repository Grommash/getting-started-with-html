Basics of the web and HTML

1. "www" or world wide web is a collection of HTML documents

2. HTML stands for hyper text markup language

3. Hyper links are connections to a specific HTML document, shortly known just as - links.

4. Using a computer with a browser software installed a person can access files on the internet
(usually represented as a vast cloud of information) which are stored on servers across the globe.

5. HTML is made of
    1) What a user can see - text content
    2) What it actually looks like - markup
    3) References to other documents - e.g. images or videos
6. Plain text in HTML is shown as plain text.




HTML markup nad documents

HTML markup is made of tags
< NAME > CONTENTS < /NAME >
The content is surrounded by an opening and a closing tag. Each tag has a name that the sotware recognizes.
A slash is always in front of the closing tag. Some tags are void tags so they don`t require a closing tag.
The entire line can be called an element.
HTML tags are various and provide a vast array of options for users.
You can create invisible boxes to organise texts, add links, contain additional contents and so on.
The HTML document consists of html tag surrounding the whole document, the head which has meta-data,
the title and the body of the html.
